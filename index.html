"git-daily" is a tool which helps you to do daily workflow easier. Using a branching model for gitflow, improve on:

  • Specialized release managements for a web application,
  • Support your remote operations.

Ver. 0.1.6: Download .zip

Ver. 0.1.6: Download .tar

Tweet

Usage

git daily init
git daily config
git daily push
git daily pull
git daily release open
git daily release list
git daily release sync
git daily release close
git daily hotfix  open
git daily hotfix  list
git daily hotfix  sync
git daily hotfix  close
git daily version
git daily help

Dependencies

Git >= 1.7.0
PHP >= 5.2.0

Install

If you are not using PHP, install PHP CLI version by apt first (on Debian or Ubuntu): 

$ sudo apt-get install php5-cli php-pear

You can install via Openpear 

$ pear channel-discover openpear.org
$ pear install openpear/Git_Daily

Download Sources

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/sotarok/git-daily
